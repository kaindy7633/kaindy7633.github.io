二、注册与充值（实测 $10 = ¥67.77）#

2.1 注册账号#

打开 runpod.io，点右上角 Sign Up。

支持邮箱注册和 Google/GitHub 快捷登录。整个过程不用实名、不用审核，填完邮箱验证码就进控制台了。

2.2 首次充值#

进入 Billing 页面，最低充值金额是 $10.00。

我实测用支付宝付款：$10.00 美元 → 人民币实付 ¥67.77（2026年6月汇率）。

⚠️ 注意：RunPod 充值不像国内平台秒到账。我第一次充值等了大约 2 分钟才显示余额，别急，刷新一下就好。

充值到账后，你会在右上角看到账户余额。确认余额 ≥ $10 就可以开始租 GPU 了。

三、选择模板和 GPU#

这是新手最容易栽跟头的一步。我来帮你省时间。

3.1 GPU 选择：首选 L40S#

RunPod 上的 GPU 从 $0.39/h 的 RTX 4000 Ada 到 $2.99/h 的 H100 都有。

跑 AI 视频推荐用 L40S（48GB 显存）——$0.86/小时。

理由很简单：

  • 48GB 显存够跑大多数视频模型（LTX、Wan2.1、CogVideoX 等）
  • $0.86/h 在 48GB 级别里性价比最高（对比 A6000 往往 $1.2+/h）
  • 社区模板 p4f6rm9tb4 已验证稳定可用

如果你只是跑图生视频、1080p 以下的输出，L40S 完全够用。没必要上来就冲 A100/H100，先跑通了再升级。

3.2 模板选择#

RunPod 的 Pod Templates 相当于预装好的环境镜像，直接选对模板可以省掉 90% 的环境配置时间。

推荐模板 ID：p4f6rm9tb4（实测稳定版）

这个模板预装了 CUDA、PyTorch 和 ComfyUI 依赖，开箱就能跑。

避坑提醒：模板 xcn7nnj1zt 多次测试 uptime=0（Pod 启动后立即挂掉），不要用。社区模板质量参差不齐，选之前看下运行次数和评分。

3.3 创建 Pod#

  1. 左侧菜单 → Pods → 点击 Deploy
  2. Select GPU → 找到 L40S，选择 Secure Cloud（不要选 Community Cloud，稳定性差很多）
  3. Template → 输入 p4f6rm9tb4 并选择
  4. 容器磁盘 → 这里一定要改！ 默认只有 20GB，跑视频模型根本不够。拉到 50GB 或更高（按 $0.03/GB/月计费，多花不了几毛钱）
  5. 其他保持默认，点击底部 Deploy 按钮

Pod 启动一般需要 1-3 分钟。等状态变成绿色的 Running 就说明好了。

四、ComfyUI 环境搭建#

Pod 启动后，你需要连接到 ComfyUI。

4.1 获取访问地址#

在 Pod 列表中，点击你刚创建的 Pod，往下翻到 Connect 区域：

  • HTTP Service: 3000 → 这就是 ComfyUI 的端口
  • 点击旁边的 Connect 按钮（或直接复制公网 URL）

在浏览器打开这个地址，你应该能看到 ComfyUI 的节点式工作界面。

4.2 ComfyUI-Login（临时 Token）#

RunPod 的 ComfyUI 模板通常启用了认证保护。首次打开会跳转到 ComfyUI-Login 页面。

解决方法：

  1. 回到 Pod 详情页，点击 View Logs
  2. 在日志中搜索 token 或 password
  3. 复制那个临时 token 登录

💡 如果没有找到 token，用 SSH 连进去运行 cat /comfyui/comfy_login/token.txt 也能拿到。

登录后进入 ComfyUI 主界面，记得先装 ComfyUI Manager（预装的模板一般都有了，如果没有，在终端运行官方安装脚本即可）。

4.3 下载第一个模型（LTX2.3）#

LTX Video 2.3 是目前性价比最高的文本生成视频模型之一——速度快、显存友好、效果不错。

在 ComfyUI Manager 中点击 Install Models，搜索 ltx，下载 ltx-video-2b-v0.9.3.safetensors。

模型下载完成后，放入 ComfyUI/models/checkpoints/ 目录。

📦 如果模板没有自带，也可以用 huggingface-cli download 命令手动拉取。

五、跑通第一个视频（LTX2.3 实测）#

5.1 加载工作流#

RunPod 模板里的 ComfyUI 可能自带 LTX 示例工作流。如果没有，去 ComfyUI Workflows 搜 “LTX text-to-video” 下载 JSON。

关键注意：ComfyUI 有两种工作流格式——API format 和 Editor format。大部分从网上下载的 workflow 是 API format，需要先在 ComfyUI 的菜单里点 Workflow → Import (API format) 导入，而不是直接拖拽。

5.2 参数设置#

导入后调整以下核心参数：

参数值说明
分辨率768×384横向 16:9 缩小版
总帧数121约 5 秒 @ 24fps
帧率24 fps标准电影帧率
Steps20-30太低画面糊，太高浪费
CFG3.0-5.0LTX 推荐区间

提示词示例：

A young woman walks through a sunlit forest, leaves falling gently around her, cinematic lighting, shallow depth of field, 4K quality

5.3 实测结果#

点击 Queue Prompt，等待生成。

L40S 实测数据：

  • 5 秒视频（121 帧 / 768×384 / 24fps）：耗时约 2-3 分钟
  • GPU 利用率：接近 100%
  • 显存占用：约 28-35GB

视频输出流畅度不错，低动态场景（人物缓慢行走、风景平移）效果最佳。

🎯 新手建议：先用低动态场景验证流程（人物走路、风景运镜），跑通了再挑战高动态（快速运动、多主体交互）。

六、成本控制——$0.86/h 怎么省到极致#

6.1 核心铁律：用完必须删 Pod#

RunPod 的计费逻辑是 按秒计费，但前提是你 真正在计费。很多人犯的错误是：Pod 在 Running 状态但 GPU 空闲，照样按 $0.86/h 扣钱。

正确的操作流程：

  1. 用完就 Stop Pod（不是 Pause，Pause 可能还会计部分费用）
  2. 到 Billing 页面确认 currentSpendPerHr = $0.00
  3. 确认余额不再下降后再关网页

⚠️ 等你下次要用时重新 Start 就行，模板和模型都在（磁盘数据不丢，只收磁盘费 $0.03/GB/月）。

6.2 成本估算#

按 L40S $0.86/h 算：

场景用量花费
快速测试30 分钟$0.43 ≈ ¥2.9
跑一个短视频1 小时$0.86 ≈ ¥5.8
生成 10 个片段2-3 小时$1.72-$2.58 ≈ ¥12-17
一整天工作8 小时$6.88 ≈ ¥46.6

对比一下：租一台 A100 物理服务器一天起步 ¥200+。RunPod 的价格对于独立创作者来说相当友好。

6.3 省钱三招#

  1. 先在本地准备好 Prompt 和工作流：别让 GPU 空转，上线直接跑
  2. 先跑低分辨率验证：用 512×256 快速试跑，确认效果再升分辨率
  3. 批量跑：把多个任务排好队列一块跑，避免反复 Stop/Start

七、避坑指南（血泪总结）#

以下每一个坑都是实测踩出来的，帮你省时间省钱：

🔴 坑1：容器盘默认只有 20GB#

创建 Pod 时如果没手动调磁盘大小，模型下载到一半就会爆盘。务必拉到 50GB 以上。

🔴 坑2：模板选错 = 从头再来#

xcn7nnj1zt 这个模板我测了 4 次，每次 uptime=0（启动即挂）。选模板前看一眼运行记录，优先选有几百次成功运行的。

🔴 坑3：ComfyUI 认证 Token#

很多 RunPod 模板默认开了 ComfyUI-Login 认证。找不到 Token 就去看 Pod Logs 或进终端手动 cat。

🔴 坑4：Workflow 格式不兼容#

网上下载的 workflow JSON 直接拖进 ComfyUI 没反应？大概率是 API format。用菜单 Import (API format) 导入即可。

🔴 坑5：忘记关 Pod#

这是我犯过的最大错误。 跑完视频去吃饭，回来发现 Pod 跑了 6 小时——$5.16 飞了。养成习惯：跑完就 Stop，去 Billing 确认 currentSpendPerHr=0。

🔴 坑6：Secure Cloud vs Community Cloud#

Community Cloud 便宜 30-50%，但 GPU 随时可能被收回，稳定性堪忧。正经跑任务请选 Secure Cloud。 省那点钱不够折腾的。

八、总结#

完整流程回顾#

注册 RunPod → 充值 $10（≈¥68）
    → 创建 Pod（L40S / 模板 p4f6rm9tb4 / 磁盘 ≥50GB）
    → 等待 Running → 获取 ComfyUI URL
    → 登录（找 Token）→ 下载 LTX 模型
    → 导入工作流 → 调参 → Queue Prompt
    → 出视频 → **立即 Stop Pod** ✅

适合谁？#

  • 🎬 独立视频创作者，想用 AI 降低制作成本
  • 🧪 AI 视频工具测评，需要快速切换不同 GPU/模型
  • 🏠 本地显卡不够用（< 24GB 显存），又不想攒机
  • 💼 有短期项目需求，不想长期租服务器

下一步#

跑通了 LTX 的文本生成视频后，你可以尝试：

  • Wan2.1：更高画质的图生视频
  • CogVideoX-5B：中文提示词友好
  • ComfyUI + AnimateDiff：更精细的运动控制

RunPod 的灵活性在于——同一个 Pod，你装什么模型就跑什么。等你熟练了这套流程，A100/H100 跑复杂项目的时间线也可以套用。
